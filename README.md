# QA Python Automation Project

Проект по автоматизации тестирования API и UI.

## Технологии
* **Python 3.11+**
* **Pytest** (фреймворк для тестирования)
* **Requests** (для API тестов)
* **Playwright** (для UI тестов)
* **Allure** (для красивых отчетов)

## Архитектура
В проекте реализован паттерн Page Object Model (POM):

 * Логика взаимодействия с элементами страниц вынесена в отдельные классы в директории pages/.

 * Это позволяет разделять описание интерфейса и саму логику тестов, делая код чище и упрощая его поддержку.

## Фикстуры (Fixtures)
 * В проекте активно используются фикстуры Pytest для управления жизненным циклом тестов:

 * Подготовка среды: Автоматическое создание сессий, настройка браузера и контекста Playwright.

 * Управление данными: Фикстуры подготавливают необходимые тестовые данные и очищают их после завершения тестов (Setup/Teardown).

 * Конфигурация: Вынос общих настроек в conftest.py позволяет переиспользовать код между разными модулями (API и UI) без дублирования.

Команды для запуска (Windows):
1. pytest --alluredir=allure-results
2. allure serve allure-results
