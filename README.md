QA Automation Practice: API & UI (Wildberries + ReqRes)
Проект представляет собой комплексный фреймворк для автоматизированного тестирования. Включает в себя интеграционные тесты API и сквозные (E2E) тесты пользовательского интерфейса.

🚀 Основные фичи
UI Automation: Глубокое тестирование маркетплейса Wildberries.

API Testing: Тестирование REST API сервиса ReqRes.

Anti-Bot Bypass: Использование Stealth-технологий (JS-инъекции, кастомные User-Agent) для обхода защиты от автоматизации.

Reports: Интерактивные отчеты Allure с вложенными скриншотами ошибок.

🛠 Технологический стек
Python 3.14+

Pytest — основной движок тестов.

Playwright — современный движок для UI-автоматизации (быстрее и стабильнее Selenium).

Requests — библиотека для работы с HTTP-запросами.

Allure Report — визуализация результатов.

🏗 Архитектура проекта (POM)
Проект построен по принципу Page Object Model, что минимизирует дублирование кода:

pages/ — локаторы и методы взаимодействия со страницами.

tests/ui/ — сценарии пользовательского интерфейса.

tests/api/ — тесты серверной логики.

conftest.py — глобальные фикстуры и конфигурация окружения.

🛒 Кейсы тестирования Wildberries
Реализована стабильная автоматизация для динамичного сайта WB:

Поиск товаров: Проверка работы поисковой строки на разных языках (Кириллица/Латиница).

Валидация выдачи: Проверка корректности отображения карточек товаров.

Корзина: Сложный сценарий добавления товара, обход капчи (закрытие поп-апов) и проверка счетчика корзины.

Особенность: Использование гибких селекторов для элементов, которые меняют свои классы динамически.

🚦 Запуск проекта
Предварительные требования
Установленный Python 3.11+.

Установленный Java JDK (для работы Allure).

Установленный Scoop (рекомендуется для Windows).

Установка
Bash
# Клонировать репозиторий
git clone https://github.com/StreetL1fter/qa-automation-practice.git
cd qa-automation-practice

# Создать и активировать окружение
python -m venv venv
source venv/bin/scripts/activate  # Для Windows: venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt
playwright install chromium
Выполнение тестов
Bash
# Запустить все тесты и собрать результаты
python -m pytest --alluredir=allure-results

# Генерировать и открыть отчет Allure
allure serve allure-results
📊 Отчетность
Благодаря интеграции с Allure, каждый запуск генерирует детальный дашборд:

Статистика прохождения (Pass/Fail).

Группировка тестов по категориям (API/UI).

Скриншоты: В случае падения UI-теста, скриншот страницы автоматически прикрепляется к отчету.
